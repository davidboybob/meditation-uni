# 카카오톡 일반채팅 연동 방안 검토 (2026-07)

> 목적: 운영자 대시보드의 리마인드/공지/정산 결과를 **카카오톡 일반 채팅방**과 연결하는 방법 모색.
> 결론 먼저: **일반 채팅방에 "자동으로" 메시지를 보내는 공식 API는 존재하지 않는다.**
> 따라서 단기는 ①복사 버튼(구현 완료) + ②카카오 공유하기 버튼, 규모가 커지면 ③알림톡(1:1 공식 발송)으로 가는 것이 정석이다.

---

## 1. 선택지 비교

| # | 방법 | 일반채팅방 지원 | 자동화 | 비용/요건 | 리스크 | 판정 |
|---|---|---|---|---|---|---|
| 1 | **리마인드 문구 복사 버튼** (대시보드 구현 완료) | ✅ (붙여넣기) | 반자동 | 0원 | 없음 | ✅ **현행 채택** |
| 2 | **카카오 공유하기 (JS SDK `Kakao.Share`)** | ✅ (사용자가 대화방 선택) | 반자동 | 무료, 카카오 개발자 앱 등록+JS키+도메인 등록 | 없음 | ✅ **다음 단계 추천** |
| 3 | **메시지 API — 나에게 보내기** | ❌ (본인 카톡만) | 자동 | 무료, 개발자 앱 | 없음 | 운영자 개인 알림용으로만 유용 |
| 4 | **메시지 API — 친구에게 보내기** | ❌ (1:1, 채팅방 불가) | 자동 | 앱 검수 + 상대 동의 필요, 월 발송 한도 | 제약 많음 | 보류 |
| 5 | **알림톡 (비즈메시지, 공식)** | ❌ (채팅방이 아닌 멤버 개별 1:1) | ✅ 완전 자동 | **사업자등록 필요**, 카카오톡 채널 + 딜러사(솔라피·알리고 등), 건당 약 7~15원, 템플릿 사전 심사 | 낮음 | 🔜 **규모 확장 시 채택** |
| 6 | **오픈채팅 봇 (공식)** | ❌ (오픈채팅 전용) | 자동 | 오픈채팅방으로 전환 필요 | 기능 제한(환영/알림/자동응답) | 방을 오픈채팅으로 바꿀 경우만 |
| 7 | **비공식 봇 (일반채팅 매크로/DB봇)** | ✅ | 자동 | 무료 | ❌ **약관 위반 — 계정 영구정지 위험** | 🚫 사용 금지 |

## 2. 권장 로드맵

### 지금 (Phase A — 구현 완료)
- 대시보드 **"리마인드 문구 복사"**: 미제출자 명단 + 오늘 본문 + 마감 안내를 클립보드로 → 카톡방 붙여넣기.
- 정산 화면 **"공유 텍스트"**: 월 정산 결과를 카톡 공지 포맷으로 복사.
- 운영 부담: 하루 1회 붙여넣기 수준. 자동화 대비 리스크 0.

### 다음 — ✅ 버튼 구현 완료 (2026-07-02, 키만 넣으면 활성화)
- **카카오 공유하기 버튼**: 대시보드(리마인드)·정산(공유)에 구현돼 있으며, 키가 없으면 자동 숨김.
- **활성화 절차**:
  1. [Kakao Developers](https://developers.kakao.com) → 앱 생성 → 앱 키에서 **JavaScript 키** 복사
  2. 앱 설정 → 플랫폼 → Web에 `https://muksang-console.vercel.app` 도메인 등록
  3. Vercel `muksang-console` 프로젝트에 환경변수 `VITE_KAKAO_JS_KEY=<JS키>` 추가 후 재배포 (`vercel --prod`)
  4. 콘솔에 "💬 카카오로 공유" 버튼이 나타남 — 클릭 시 대화방 선택 UI로 일반 채팅방에 바로 전송

### 나중 (규모 확장 시)
- **알림톡**: 참여자가 많아지고 "개별 리마인드"가 필요해지면 도입. 요건: 사업자등록 → 카카오톡 채널 개설·비즈니스 인증 → 딜러사(예: [SOLAPI](https://solapi.com/guides/kakao-ata-guide), 알리고 등) 계약 → 템플릿 심사 → API 발송. 마감 1시간 전 미제출자에게 자동 개별 발송 같은 시나리오 구현 가능. Supabase Edge Function + pg_cron으로 트리거.
- 참고: 알림톡은 "채팅방"이 아니라 **멤버 개인에게 1:1**로 가는 정보성 메시지다. 채팅방 문화 유지가 중요하면 ①/②를 계속 쓰는 것이 맞다.

## 3. 하지 말아야 할 것
- 일반채팅방 자동 봇(안드로이드 매크로, 세션 후킹 등)은 카카오 운영정책 위반으로 **개인 계정 영구정지** 사례가 보고됨. 커뮤니티 운영 계정으로는 절대 금지.
