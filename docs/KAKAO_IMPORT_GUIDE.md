# 카톡 대화 → 자동 출석 (사용 가이드)

> 운영자가 카톡방 대화를 내보내 콘솔에 붙여넣으면, 각 메시지의 **날짜 + 그날 통독 본문(예: 에스겔 3장)**을
> 대조해 출석을 자동으로 매칭합니다. **블로그 링크·채팅 문구 모두** 인식합니다.
>
> ⚠️ 왜 실시간 자동이 아닌가: 카카오톡은 **일반 채팅방을 외부 앱이 실시간으로 읽는 공식 API가 없습니다**
> (비공식 봇은 계정 영구정지 위험). 그래서 안전한 방법은 "대화 내보내기 → 붙여넣기"입니다. 하루 1번, 1분이면 됩니다.

## 준비: 통독 일정부터 등록 (1회/주기)
콘솔 **통독 일정**에서 날짜별 본문을 등록해 두세요. 자동 생성기로 "에스겔 1장, 시작일 …"만 넣으면 하루 1장씩 채워집니다.
→ 매칭은 이 일정의 `날짜=본문`을 기준으로 이뤄집니다.

## 1단계 — 카톡 대화 내보내기

### 📱 아이폰
1. 채팅방 열기 → 오른쪽 위 **≡** → 하단 **⚙ 설정 아이콘** → **대화 내용 내보내기**
2. **텍스트 메시지만 보내기**(또는 이메일로 내보내기) 선택 → 나에게 메일/메모로 보내기
3. 받은 텍스트 전체 복사

### 🤖 안드로이드
1. 채팅방 열기 → 오른쪽 위 **≡** → **⚙ 설정** → **대화 내용 내보내기** → **텍스트만 저장**
2. 저장된 `.txt` 파일을 열어 전체 복사(또는 파일 앱에서 내용 열기)

### 💻 PC (카카오톡 데스크톱)
1. 채팅방 → 상단 메뉴 → **대화 내용 → 대화 내용 내보내기(.txt)**
2. 저장된 파일을 메모장으로 열어 전체 복사

## 2단계 — 콘솔에 붙여넣고 분석
1. 콘솔 → **카톡 출석** 메뉴 → 큰 입력창에 복사한 대화 **전체 붙여넣기**
2. **분석하기** 클릭 → "메시지 N건 · 발신자 M명 → K건 매칭" 표시
3. **매칭 결과** 표에서 이름/날짜/출석·지각/근거(🔗링크 또는 💬문구) 확인
4. 이상 없으면 **출석으로 반영** → 출석부·정산에 즉시 집계

## 이름이 매칭 안 될 때
카톡 표시명이 콘솔 멤버 이름과 다르면 "매칭 안 된 카톡 이름"에 뜹니다.
- 즉석에서 드롭다운으로 멤버와 연결 → **다시 분석**
- **멤버 관리**에서 각 멤버의 "카톡 이름"을 저장해두면 **다음부터 자동 매칭**됩니다(한 번만 해두면 됨).

## 지각 판정
그룹 **설정 → 지각 판정 시각**(HH:MM)을 넣어두면, 그 시각 이후 메시지는 자동으로 **지각**으로 매칭됩니다. 비워두면 전부 출석 처리.

---

## 📋 권장: 멤버 인증 양식 (자동 매칭 정확도 ↑)

멤버들에게 아래처럼 올려달라고 안내하면 매칭이 거의 100% 됩니다. **핵심은 "책 이름 + 장"을 꼭 포함**하는 것.

**방식 1 — 블로그 링크형**
```
[에스겔 3장] 오늘의 묵상
https://blog.naver.com/내블로그/글주소
한 줄 나눔: 파수꾼의 사명을 다시 새겼습니다.
```
> 링크만 달랑 올리면 매칭이 어렵습니다. **"에스겔 3장"을 링크와 함께** 적어주세요.

**방식 2 — 채팅 문구형**
```
에스겔 3장 ✅
기도제목: 말씀에 순종
오늘의 질문: 나는 어떤 파수꾼인가?
```

### 매칭이 잘 되는 표기 (모두 인식)
`에스겔 3장` · `겔 3장` · `겔3장` · `에스겔 3` · `에스겔 3:1-5` · `[에스겔 3장]`
→ 정식명·표준 약자·붙여쓰기·장 생략·절 범위 모두 인식합니다(성경 66권 전체).

### 피해야 할 표기
- 장 번호 없이 책만: `에스겔 묵상` ✗ (몇 장인지 모름)
- 다른 날 본문 소급: 그날 통독 일정과 다른 장을 쓰면 매칭 안 됨(정상)

## 자주 묻는 질문
- **Q. 사진/이모티콘만 올린 건요?** → 텍스트가 없어 매칭 대상이 아닙니다. 본문 표기가 있는 메시지만 인정.
- **Q. 한 명이 하루에 여러 번 올리면?** → 그날 본문과 맞는 첫 메시지 1건으로 출석 1회 인정(중복 없음).
- **Q. 이미 앱에서 '내 묵상' 제출한 멤버는?** → 같은 날짜는 덮어쓰기(중복 방지). 카톡·앱 어느 쪽이든 1회 인정.
- **Q. 실수로 반영했어요.** → 출석부에서 해당 셀을 눌러 수정/삭제할 수 있습니다.
